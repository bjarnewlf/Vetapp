Das Feature

Chat-basierter
Tiergesundheits-Assistent

★  Premium-Feature
💬
Natürlichsprachiger Chat
User stellen Fragen in natürlicher Sprache — keine Formulare, keine Checkboxen. Der Assistent antwortet kontextuell und auf Deutsch.
🐶
Persönlicher Tier-Kontext
Der Assistent kennt Name, Rasse, Alter, Gewicht und Impfstatus des Tieres. Antworten sind individuell — nicht generisch.
⚡
Powered by Claude Haiku
Anthropic Claude Haiku — schnell, kosteneffizient, präzise. API-Key bleibt serverseitig (Supabase Edge Function), nie im Client.
🔒
Premium-Gate
Kostenloser Teaser auf dem Profil-Screen. Vollzugriff nur für Pro-User. Free-User sehen eine Paywall mit klarem Mehrwert-Versprechen.
⚠️ Wichtiger Hinweis: Der KI-Assistent ist kein Ersatz für eine tierärztliche Untersuchung. Bei ernsthaften Symptomen immer einen Tierarzt aufsuchen. Dieser Hinweis erscheint prominent zu Beginn jeder Chat-Session.

Technik

Technische Architektur

📱
React Native App
ChatScreen + aiService.ts
HTTPS + JWT-Auth
⚙️
Supabase Edge Function
ai-chat / Deno-Runtime
Anthropic SDK
🤖
Claude Haiku API
Anthropic • claude-haiku-4-5
🔑
API-Key nur serverseitig
Der Anthropic-API-Key liegt ausschließlich in der Edge Function als Umgebungsvariable. Kein Zugriff über den Client möglich.
🛡️
JWT-Authentifizierung
Jeder Aufruf der Edge Function erfordert ein gültiges Supabase-JWT. Unauthentifizierte Anfragen werden abgelehnt.
🐾
Pet-Daten als System-Prompt
Name, Rasse, Alter, Gewicht, Impfstatus werden in den System-Prompt injiziert. Das Modell "kennt" das Tier ohne separate DB-Abfrage.
💰
Claude Haiku = kosteneffizient
Haiku ist ~25x günstiger als Opus bei vergleichbarer Qualität für medizinische Kurztexte. Ideal für Chat-Anwendungen.
