Create a **mobile app UI prototype** for a pet management app called **VetApp**. The app is designed for **pet owners who want to organize all important information about their pets in one place**.

The app should be **simple, modern, friendly, and easy to use**, with a clean layout and large touch-friendly buttons. The design should feel like a modern lifestyle app for pet owners.

Use **card-based layouts**, large pet images, rounded corners, and clear typography. The interface should feel warm, friendly, and trustworthy.

Navigation:
Use a **bottom navigation bar with 5 tabs**:
Home
My Pets
Reminders
Vet Contact
Profile

Create the following **mobile screens**:

1. Onboarding Screen
   Headline: “All your pet’s important information in one place.”
   Short explanation text about organizing pet health and reminders.
   Button: “Get Started”.

2. Home / Dashboard
   Greeting text.
   Cards showing the user’s pets (image + name).
   Section showing the **next upcoming reminder**.
   Button: “Add Pet”.

3. Add Pet Screen
   Form with the following fields:

* Pet Name
* Animal Type
* Breed
* Birth Date
* Microchip Code
* Upload Pet Photo

Primary button: “Save Pet”.

4. Pet Profile Screen
   Large pet photo.
   Pet information section with:

* Name
* Breed
* Age
* Microchip Code

Below that, three sections:

* Treatments
* Reminders
* Vet Contact

Buttons:
“Add Treatment”
“Add Reminder”

5. Treatments Screen
   List of treatments in card format showing:

* Treatment name
* Date
* Notes

Button: “Add Treatment”.

6. Add Treatment Screen
   Form fields:

* Treatment name
* Date
* Notes

Button: “Save”.

7. Reminders Screen
   List of upcoming reminders showing:

* Reminder title
* Date
* Status

Button: “Add Reminder”.

8. Add Reminder Screen
   Form fields:

* Reminder title
* Date
* Description

Button: “Save”.

9. Vet Contact Screen
   Display vet information:

* Clinic name
* Phone number
* Address

Button: “Call Vet”.

10. Premium / AI Feature Screen
    Illustration showing a future AI feature.
    Text: “AI-powered pet health assistant – coming soon.”
    This screen previews future premium features.

Design guidelines:

* Mobile-first layout
* Clean modern UI
* Card-based components
* Friendly colors suitable for a pet app
* Rounded UI elements
* Simple forms
* Clear navigation

The prototype should allow a **basic clickable flow** from:
Home → Add Pet → Pet Profile → Add Treatment → Add Reminder.
